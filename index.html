Аббревиатура HTTP расшифровывается как HyperText Transfer Protocol, «протокол передачи гипертекста». Задача, которая традиционно решается с помощью протокола HTTP — обмен данными между веб-браузером и веб-сервером. На данный момент именно благодаря протоколу HTTP обеспечивается работа Всемирной паутины (World Wide Web).
